## ğŸ“¦ Installation

### Method 1: Manual Installation
1. Clone or download this repository
2. Copy the `themes` folder to your Zed themes directory:
   ```bash
   cp -r themes ~/.config/zed/
   ```
3. Restart Zed
4. Open settings (`Cmd+,` or `Ctrl+,`)
5. Search for "theme" and select "Super Mario World"
